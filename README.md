# camunda-bpm-sdk-js

Javascript client library for [camunda BPM](https://github.com/camunda/camunda-bpm-platform)

This respository is a fork from `https://github.com/camunda/camunda-bpm-sdk-js.git#5ba55c2e5bd18620e174c23b45aa09b830a29737` (7.10.0-SNAPSHOT).\
Camunda forms which used a deprectaed version of `angualr` were removed and the required dependecies were updated.
